# aocelm2021
Advent of Code 2021 in Elm Lang

## Running

Simple:
```
elm make src/Day01.elm
open index.html
```

Another way would be to use elm reactor and navigate to the day in src.
```
elm reactor
```

## Formatting

```
elm-format --yes src
```

## Tests

```
npm install -g elm-test
```

Running

```
elm-test
```
